:zap::globe_with_meridians::zap: Typesystem directives to declare additional validation on top of GraphQL type system.

## Introduction

This package allows server to declare additional constraints on accepted values for arguments and input fields. It is also possible for client to declare constraints for variables in request document.

## Installation

Install package using composer

```composer require infinityloop-dev/graphpinator-constraint-directives```
